# Project V 개발 일지 - Day 13

## 개발 날짜

2026년 7월 14일

## 개발 목표

히로인이 낮은 체력에서 자신을 회복할 수 있는 행동을 구현한다.

## 구현 내용

- HeroineActionType에 Heal 행동 추가
- HeroineActionData에 Heal Amount 데이터 추가
- HA008 회복의 기도 행동 데이터 생성
- 히로인 체력 회복 처리 구현
- 최대 체력을 넘지 않는 회복량 제한 구현
- 최대 체력에서 회복 행동이 선택되지 않도록 조건 추가
- 회복 행동 예고 UI 표시
- 기존 행동 가중치 시스템과 회복 행동 연동
- 기존 쿨다운 및 연속 사용 제한 시스템과 연동

## 회복의 기도 설정

- 행동 ID: HA008
- 행동 종류: Heal
- 대상: Self
- 회복량: 5
- 가중치: 20
- 쿨다운: 3턴
- 최대 연속 사용: 1회
- 사용 체력 구간: 0% 이상 50% 이하

## 낮은 체력 행동 가중치

- 절박한 휩쓸기: 30
- 절박한 일격: 25
- 마력 장벽: 25
- 회복의 기도: 20

## 테스트 결과

- 최대 체력에서 회복 행동이 선택되지 않음
- 체력 50% 이하에서 회복 행동이 후보에 포함됨
- 회복 후 체력이 정상적으로 증가함
- 회복 후 체력이 최대 체력을 넘지 않음
- 실제 회복량이 결과 UI에 표시됨
- 회복 행동의 쿨다운이 정상 적용됨

히로인 행동에 상태 효과를 추가할 수 있도록 상태 효과의 기본 구조를 설계한다.
