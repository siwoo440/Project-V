\# Project V 개발 일지 - Day 20



\## 개발 날짜



2026년 7월 15일



\## 개발 목표



전투 중 발생한 피해, 상태 효과, 턴 전환과 승패를 시간순으로 확인할 수 있는 전투 로그 패널을 구현한다.



\## 구현 내용



\- BattleLogCategory 로그 분류 추가

\- BattleLogUI 전투 로그 관리 스크립트 구현

\- 전투 로그 Scroll View UI 제작

\- 턴 번호와 로그 분류 표시

\- 로그 분류별 색상 표시

\- 최대 50개 로그 보관

\- 오래된 로그 자동 제거

\- 최신 로그 자동 스크롤

\- 마물 공격 및 상태 효과 로그 연동

\- 히로인 공격, 회복, 보호막 행동 로그 연동

\- Poison 지속 피해 로그 연동

\- Purification 정화 로그 연동

\- 카드 사용 및 마물 소환 로그 연동

\- 턴 시작, 종료 및 전투 결과 로그 연동



\## 로그 분류



\- System: 전투 시작, 턴 전환, 승패

\- Player: 카드 사용, 마물 소환, 마물 공격

\- Heroine: 공격, 회복, 보호막 행동

\- Status: 상태 효과 적용, 독 피해, 정화



\## 로그 표시 형식



\- 턴 번호: T01, T02 형식

\- 로그 분류: System, Player, Heroine, Status

\- 여러 줄 결과는 한 줄로 변환

\- 최신 로그는 패널 가장 아래에 표시
