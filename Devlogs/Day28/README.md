\## 성장 규칙



\- 신규 마물은 레벨 1과 보유 수량 1로 등록한다.

\- 중복 마물은 보유 수량이 증가한다.

\- 중복 포획 시 추가 경험치를 받는다.

\- 전투 경험치는 현재 보유한 모든 마물에게 지급한다.

\- 필요 경험치는 현재 레벨에 따라 증가한다.

\- 레벨업 능력치는 MonsterData의 성장값을 사용한다.

\- 보유 데이터가 없는 마물은 기본 능력치를 사용한다.
