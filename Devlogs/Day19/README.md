\# Project V 개발 일지 - Day 19



\## 개발 날짜



2026년 7월 15일



\## 개발 목표



상태 효과 아이콘에 마우스를 올렸을 때 상세 정보를 보여주는 툴팁을 구현한다.



\## 구현 내용



\- StatusEffectData에 상세 설명 데이터 추가

\- 상태 효과 툴팁 패널 제작

\- 상태 이름, 분류, 설명, 지속시간 UI 구성

\- StatusEffectTooltipUI 구현

\- 상태 효과 아이콘 포인터 진입 이벤트 구현

\- 포인터 이동에 따른 툴팁 위치 갱신 구현

\- 포인터 이탈 시 툴팁 숨김 구현

\- 화면 경계를 벗어나지 않는 위치 보정 구현

\- 버프와 디버프 분류 표시

\- 상태 효과 종류별 수치 설명 구현

\- 상태 정화 및 만료 시 툴팁 자동 제거



\## 툴팁 표시 정보



\- 상태 효과 이름

\- Buff 또는 Debuff 분류

\- 상태 효과 상세 설명

\- 실제 적용 수치

\- 남은 지속시간

\- 지속시간 감소 기준



\## 상태 효과 설명



\- DefenseUp: 방어력 증가

\- AttackDown: 히로인 공격 피해 감소

\- Poison: 히로인 행동 시작 전 직접 HP 피해



\## 테스트 결과



\- 상태 아이콘에 마우스를 올리면 툴팁 표시

\- 마우스 이동 시 툴팁 위치 갱신

\- 아이콘에서 벗어나면 툴팁 숨김

\- 화면 경계 밖으로 툴팁이 나가지 않음

\- 버프와 디버프가 올바르게 구분됨

\- 상태별 설명과 수치가 정상 표시됨

\- 정화 또는 만료 시 아이콘과 툴팁이 함께 제거됨



\## 다음 개발 방향



전투 중 발생한 피해, 보호막, 상태 효과 적용과 해제를 시간순으로 확인할 수 있는 전투 로그 패널을 구현한다.
