# Project V 개발 일지 - Day 22

## 개발 날짜

2026년 7월 15일

## 개발 목표

Project V의 두 번째 전투 수치인 히로인 성욕 게이지 시스템의 기반을 구현한다.

## 구현 내용

- MonsterData에 Lust Damage 수치 추가
- MonsterUnit에 성욕 피해 프로퍼티 추가
- 마물 UI에 성욕 피해 표시
- BattleManager에 최대 성욕 수치 추가
- 마물 공격 시 히로인 성욕 증가
- 성욕 수치가 최대치를 넘지 않도록 제한
- 실제 적용된 성욕 증가량 계산
- 성욕 게이지 Slider 구현
- 성욕 최대치 MAX 문구 표시
- 공격 결과에 성욕 증가량 표시
- Player 전투 로그에 성욕 증가량 기록

## 전투 처리 방식

현재 마물이 공격하면 히로인 HP 피해와 성욕 피해가 함께 적용된다.

성욕 수치는 0부터 시작하며 최대 100까지만 증가한다.

성욕이 최대치에 도달해도 22일차에는 전투가 종료되지 않는다.

## UI 구성

- HeroineLustText
- HeroineLustSlider
- MonsterLustDamageText

## 테스트 결과

- 전투 시작 시 성욕 0으로 초기화
- 마물 공격 시 히로인 HP 피해 적용
- 마물 공격 시 히로인 성욕 증가
- 마물 데이터별 성욕 피해 정상 적용
- 성욕 Slider 정상 갱신
- 성욕 수치 최대 100 제한
- 최대치 도달 시 MAX 문구 표시
- 결과 문구에 성욕 증가량 표시
- 전투 로그에 성욕 증가량 기록
