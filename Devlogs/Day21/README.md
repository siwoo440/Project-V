\- 전체 로그 필터 구현

\- 시스템 로그 필터 구현

\## 버그 수정



\### 필터 변경 시 첫 턴 로그가 보이지 않는 문제



`All` 필터에서 다른 필터로 변경하면 스크롤이 자동으로 맨 아래로 이동하여 첫 턴 로그가 화면에 보이지 않는 문제가 발생했다.



필터를 변경할 때 `RequestScrollToBottom()`이 실행되는 것이 원인이었다.



필터 변경 시 `RequestScrollToTop()`을 실행하도록 수정하여 해당 필터의 첫 번째 로그부터 표시되도록 처리했다.



\### 첫 번째 플레이어 턴 시작 로그 누락



두 번째 턴부터는 `Player turn started.` 로그가 기록되지만, 전투 시작 직후의 첫 번째 플레이어 턴에는 해당 로그가 기록되지 않았다.



`InitializeBattle()`에 첫 번째 플레이어 턴 시작 로그를 추가했다.
