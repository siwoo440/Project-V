\# Project V 개발 일지 - Day 14



\## 개발 날짜



2026년 7월 14일



\## 개발 목표



히로인 행동에 일정 턴 동안 유지되는 상태 효과 구조를 구현한다.



\## 구현 내용



\- StatusEffectType 상태 효과 열거형 추가

\- StatusEffectData ScriptableObject 추가

\- ActiveStatusEffect 런타임 클래스 추가

\- HeroineActionType에 ApplyStatus 추가

\- HeroineActionData에 적용 상태 효과 데이터 추가

\- 히로인 활성 상태 효과 목록 구현

\- 상태 효과 지속시간 감소 및 만료 처리 구현

\- 방어력 증가 상태 효과 구현

\- 상태 효과를 마물 공격 피해 계산에 연동

\- 동일 상태 효과 중복 적용 방지

\- 상태 효과 행동 예고 UI 구현

\- 현재 상태와 남은 지속시간 UI 구현



\## 신규 상태 효과



\- ID: ST001

\- 이름: Defense Up

\- 종류: DefenseUp

\- 증가량: 2

\- 지속시간: 플레이어 턴 2회

\- 중첩 여부: 중첩 불가



\## 신규 히로인 행동



\- ID: HA009

\- 이름: Defensive Stance

\- 행동 종류: ApplyStatus

\- 대상: Self

\- 적용 효과: ST001 Defense Up

\- 가중치: 15

\- 쿨다운: 3턴

\- 최대 연속 사용: 1회



\## 테스트 결과



\- 방어 태세 행동이 정상적으로 선택됨

\- 행동 예고에 상태 효과와 지속시간이 표시됨

\- 행동 실행 후 히로인의 방어력이 증가함

\- 증가한 방어력이 마물 공격 피해에 적용됨

\- 플레이어 턴 종료 시 지속시간이 감소함

\- 지속시간이 0이 되면 상태 효과가 제거됨

\- 상태 효과 제거 후 기본 방어력으로 복구됨

\- 적용 중에는 같은 상태 행동이 다시 선택되지 않음

마물 공격에 상태 효과 부여 기능을 추가하고 공격력 감소 또는 지속 피해 효과를 구현한다.
