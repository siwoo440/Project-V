\# Project V 개발 일지 - Day 15



\## 개발 날짜



2026년 7월 14일



\## 개발 목표



마물 공격이 히로인에게 공격력 감소 상태 효과를 부여하도록 구현한다.



\## 구현 내용



\- StatusEffectType에 AttackDown 추가

\- 상태 효과 지속시간 감소 시점 분리

\- AfterPlayerTurn과 AfterHeroineTurn 구현

\- 상태 효과 재적용 시 지속시간 갱신 구현

\- MonsterData에 공격 상태 효과 데이터 추가

\- MonsterUnit에서 공격 상태 효과 공개

\- 마물 공격과 상태 효과 적용 연동

\- 히로인 공격력 감소 계산 구현

\- 단일 공격, 광역 공격, 플레이어 직접 공격에 감소량 적용

\- 행동 예고에 현재 공격 피해 표시

\- 상태 효과 UI 증가 및 감소 부호 구분



\## 신규 상태 효과



\- ID: ST002

\- 이름: Attack Down

\- 종류: AttackDown

\- 감소량: 2

\- 지속시간: 히로인 행동 2회

\- 감소 시점: AfterHeroineTurn

\- 중첩 방식: 중첩하지 않고 지속시간 갱신



\## 마물 공격 규칙



\- Attack Status Effect가 설정된 마물만 상태 효과 부여

\- 보호막이 피해를 전부 흡수해도 상태 효과 적용

\- 동일 상태 효과 재적용 시 지속시간 초기화

\- 상태 효과가 없는 마물은 기존 공격만 실행



\## 테스트 결과



\- 지정된 마물 공격 후 Attack Down이 적용됨

\- 히로인 행동 예고 피해량이 감소함

\- 단일 공격과 광역 공격에 감소량이 적용됨

\- 플레이어 직접 공격에도 감소량이 적용됨

\- 히로인 행동 종료 후 지속시간이 감소함

\- 지속시간 0에서 상태 효과가 제거됨

\- 재적용 시 중첩되지 않고 지속시간이 갱신됨

\- 일반 마물 공격에는 상태 효과가 적용되지 않음



\## 다음 개발 방향



마물 공격이 지속 피해 상태를 부여하도록 확장하고 턴마다 체력이 감소하는 독 효과를 구현한다.
