\# Project V 개발 일지 - Day 17



\## 개발 날짜



2026년 7월 15일



\## 개발 목표



히로인이 자신에게 적용된 해로운 상태 효과를 제거하는 정화 행동을 구현한다.



\## 구현 내용



\- StatusEffectData에 해로운 상태 효과 여부 추가

\- HeroineActionType에 Cleanse 추가

\- HeroineActionData에 Cleanse Count 추가

\- 해로운 상태 효과 보유 여부 검사 구현

\- 상태 효과 정화 우선순위 구현

\- 최우선 정화 대상 검색 구현

\- 해로운 상태 효과 제거 처리 구현

\- 해로운 상태가 없을 때 정화 행동 선택 제외

\- 정화 행동 예고 문구 구현

\- 기존 가중치와 쿨다운 시스템 연동

\- 이로운 상태 효과 정화 제외 처리



\## 정화 행동 설정



\- ID: HA010

\- 이름: Purification

\- 행동 종류: Cleanse

\- 대상: Self

\- 한 번에 제거하는 상태 효과: 1개

\- 가중치: 30

\- 쿨다운: 2턴

\- 최대 연속 사용: 1회



\## 정화 우선순위



1\. Poison

2\. AttackDown

3\. 기타 해로운 상태 효과



\## 상태 효과 분류



\- DefenseUp: 이로운 상태

\- AttackDown: 해로운 상태

\- Poison: 해로운 상태



\## 테스트 결과



\- 해로운 상태가 있을 때만 Purification이 후보에 포함됨

\- Poison이 AttackDown보다 먼저 제거됨

\- 한 번에 해로운 상태 효과 하나만 제거됨

\- DefenseUp은 정화해도 유지됨

\- 정화 후 Poison 지속 피해가 중단됨

\- 정화 후 AttackDown 공격력 감소가 해제됨

\- 정화 행동에 쿨다운과 가중치가 적용됨



\## 다음 개발 방향



히로인 상태 효과에 아이콘과 색상을 추가해 현재 버프와 디버프를 시각적으로 구분한다.
