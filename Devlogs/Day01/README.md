\# Project V 개발 일지



\## 1일차: 기본 전투 화면 및 턴 전환 구현



\### 개발 목표



Unity 6 LTS를 사용하여 프로젝트 V의 기본 프로젝트를 생성하고, 플레이어 턴과 히로인 턴이 반복되는 최소 전투 구조를 구현한다.



\### 개발 환경



\- 엔진: Unity 6 LTS

\- 템플릿: Universal 2D

\- 플랫폼: Windows PC

\- UI: uGUI, TextMeshPro

\- 버전 관리: Git, GitHub



\### 구현 내용



\- 프로젝트 V Unity 프로젝트 생성

\- `\_ProjectV` 기준 폴더 구조 생성

\- `BattleScene` 전투 장면 생성

\- 플레이어 HP와 마나 UI 구현

\- 히로인 HP와 성욕 게이지 UI 구현

\- 마물 필드와 카드 손패 임시 영역 구현

\- 플레이어 턴과 히로인 턴 전환 구현

\- 턴 종료 버튼 구현

\- 히로인의 고정 피해 공격 구현

\- 다음 턴 시작 시 최대 마나 증가 구현

\- 최대 마나 10 제한 구현

\- 플레이어 HP 0 도달 시 패배 처리 구현

\- 패배 후 턴 종료 버튼 비활성화 구현

\### 테스트 결과



\- 전투가 플레이어 턴으로 시작되는 것을 확인했다.

\- 초기 플레이어 HP가 30으로 표시되는 것을 확인했다.

\- 초기 마나가 1로 표시되는 것을 확인했다.

\- 턴 종료 버튼을 누르면 히로인 턴으로 변경되는 것을 확인했다.

\- 히로인 턴마다 플레이어 HP가 3 감소하는 것을 확인했다.

\- 다음 플레이어 턴에 최대 마나가 1 증가하는 것을 확인했다.

\- 최대 마나가 10을 넘지 않는 것을 확인했다.

\- 플레이어 HP가 0이 되면 패배가 표시되는 것을 확인했다.

\- 패배 후 턴 종료 버튼이 비활성화되는 것을 확인했다.

\- Unity Console Error 0건을 확인했다.
